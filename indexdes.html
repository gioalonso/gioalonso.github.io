    Checkertool

    CheckerTool es una aplicación android usada para tener un regístro de gente invitada a un determinado evento.
    Si una persona no esta en la lista de invitados la aplicación mostrara a esta persona como no invitada, de otra manera la app mostrará un gráfico anunciando acceso autorizado a esta persona asi como tendra un regístro de la hora de llegada de dicha persona y lo guardara en un archivo de Excel.

    La tecnología implementada en esta aplicación es:

  • Android XML como Frontend.
  • Java como backend.
  • SQLite como manejo de base de datos.

    • SQL Queries




    He creado varias macros de excel para obtener información depurada de una base de datos SQL server.
    El método que he utilizado es crear macros que básicamente lo que hacen es una comunicación con la base de datos, despues por medio de un procedimiento almacenado se obtiene y depúra la información según el usuario la requiere.

    La tecnología aplicada para estas aplicaciones es:
    - SQL Server para la creación de los procedimientos almacenados.
    - VBA para el desarrollo de las macros.

    Smart Planner




    Smart Planner es básicamente una herramienta WEB creada para dar solución a una migración de sistema debido a que la compañía quería hacer un cambio de software de Microsoft Office a la plataforma de ofimática de Google, Smart Planner lo que hace de manera básica es conectarse a una base de datos MSQL y mostrar los datos en una página WEB.

    La tecnología aplicada para estas aplicaciones es:
    - SPRING como Framework. - MSQL server para manejo de base de datos. - HTML, CSS y Bootstrap como front-end. - Java como back-end.
